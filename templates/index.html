About This Project

This is a university course tracking system built for CS:4400 Database Systems at the University of Iowa. The system manages students, courses, sections, employees (professors, TAs, advisors), academic requirements, and majors.

  • Database: MySQL with triggers, views, stored procedures
  • Backend: Python Flask with raw SQL queries
  • Features: Course enrollment, prerequisite tracking, cross-listing, requirements
